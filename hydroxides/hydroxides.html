Гидроксиды — Конспект по химии

Что такое гидроксиды?

Гидроксиды — это сложные вещества, состоящие из катиона металла (или NH₄⁺) и гидроксид-аниона OH⁻.

Общая формула: Ме(OH)n, где n — валентность металла.

Классификация гидроксидов

  • Растворимые (щёлочи) — гидроксиды щелочных и щелочноземельных металлов. Пример: NaOH, KOH, Ca(OH)₂.
  • Нерастворимые основания — не растворяются в воде. Пример: Fe(OH)₂, Cu(OH)₂.
  • Амфотерные гидроксиды — проявляют как основные, так и кислотные свойства. Пример: Al(OH)₃, Zn(OH)₂, Cr(OH)₃.

Получение гидроксидов

  1. Активные металлы + вода:
    2Na + 2H₂O → 2NaOH + H₂↑
  2. Оксиды активных металлов + вода:
    CaO + H₂O → Ca(OH)₂
  3. Соль + щёлочь:
    FeCl₃ + 3NaOH → Fe(OH)₃↓ + 3NaCl

Химические свойства

Щёлочи:

  • С кислотами: NaOH + HCl → NaCl + H₂O (нейтрализация)
  • С кислотными оксидами: 2NaOH + CO₂ → Na₂CO₃ + H₂O
  • С солями: 2NaOH + CuSO₄ → Cu(OH)₂↓ + Na₂SO₄

Амфотерные гидроксиды:

  • С кислотой: Zn(OH)₂ + 2HCl → ZnCl₂ + 2H₂O
  • С щёлочью: Zn(OH)₂ + 2NaOH → Na₂[Zn(OH)₄]

Примеры важных гидроксидов

Формула Название Тип Применение
NaOH Гидроксид натрия Щёлочь Производство мыла, бумаги, очистка труб

Ca(OH)₂ Гашёная известь Малорастворимый Строительство, дезинфекция
Al(OH)₃ Гидроксид алюминия Амфотерный Антацид, осветление воды

Заключение

Гидроксиды — ключевой класс веществ в неорганической химии. Они лежат в основе понимания кислотно-основных реакций, используются в промышленности, медицине и повседневной жизни. Умение определять тип гидроксида помогает предсказать его поведение в реакциях.
