Бериллий (Be)

Физические свойства

  • Агрегатное состояние при н.у.: твёрдое вещество
  • Цвет: серебристо-белый
  • Плотность: 1.85 g/cm³
  • Температура плавления: 1287°C
  • Электропроводность: высокая
  • Теплопроводность: одна из самых высоких среди металлов

Химические свойства

Бериллий — переходный металл с низкой химической активностью. Реагирует с кислотами и при высоких температурах:

  • С соляной кислотой: Be + 2HCl → BeCl₂ + H₂↑
  • С кислородом: 2Be + O₂ → 2BeO
  • С азотом: 3Be + N₂ → Be₃N₂
  • С водой: не реагирует при комнатной температуре

История открытия

Бериллий был открыт французским химиком Луи Никола Вокуленом в 1798 году при анализе минерала берилла. Он обнаружил новое соединение, которое позже получило название оксида бериллия (BeO). В 1828 году Фридрих Вёлер и Антуан Бусси независимо друг от друга получили чистый металлический бериллий электролизом его солей.

Применение

  • Авиация и космос: легкие сплавы (Be + Cu → прочные материалы).
  • Электроника: теплоотвод в радиаторах (Be → высокая теплопроводность).
  • Ядерная энергетика: замедлитель нейтронов (Be → реакторы).
  • Медицина: X-лучевые трубки (Be → прозрачный для рентгеновских лучей).

Интересные факты

  • Бериллий — один из самых токсичных металлов при вдыхании его пыли.
  • Название происходит от латинского "beryllo" — берилл.
  • Используется в производстве сверхпроводящих магнитов (Be → компонент материалов).
  • Один из самых редких элементов на Земле (Be → образуется в звездах).
