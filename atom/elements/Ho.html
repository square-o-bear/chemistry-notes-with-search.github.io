Гольмий (Ho)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 8.795 г/см³
  • Температура плавления: 1474 °C
  • Электропроводность: низкая
  • Теплопроводность: средняя

Химические свойства

Лантанид, активно окисляется на воздухе и взаимодействует с водой:

  • С кислородом: 4Ho + 3O₂ → 2Ho₂O₃
  • С водой: 2Ho + 6H₂O → 2Ho(OH)₃ + 3H₂↑
  • С галогенами: 2Ho + 3Cl₂ → 2HoCl₃
  • С кислотами: Ho + 3HCl → HoCl₃ + 3/2H₂↑

История открытия

Открыт Пером Теодором Клеве в 1878 году при анализе оксида эрбия. Название дано в честь "Holmia", латинского названия Стокгольма.

Применение

  • Магниты: используется в неодимово-гольмиевых магнитах для улучшения термостойкости.
  • Ядерная энергетика: Ho₂O₃ применяется в поглотителях нейтронов.
  • Оптика: добавляется в люминофоры для светодиодов (красный свет).
  • Электроника: используется в лазерах и стабилизаторах напряжения.

Интересные факты

  • Гольмий обладает одним из самых высоких магнитных моментов среди всех элементов.
  • Его оксид Ho₂O₃ используется в защитных стеклах от радиации.
  • Соединения гольмия добавляются в светодиоды для создания яркого красного цвета.
