Лютеций (Lu)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 9.84 г/см³
  • Температура плавления: 1663 °C
  • Электропроводность: средняя
  • Теплопроводность: средняя

Химические свойства

Лантанид, активно взаимодействует с кислородом и водой:

  • С кислородом: 4Lu + 3O₂ → 2Lu₂O₃
  • С водой: Lu + 3H₂O → Lu(OH)₃ + 3/2H₂↑
  • С галогенами: 2Lu + 3Cl₂ → 2LuCl₃
  • С кислотами: Lu + 3HCl → LuCl₃ + 3/2H₂↑

История открытия

Открыт Габриэлом Лютециусом в 1907 году при анализе минерала церита. Название дано в честь римского бога Луциус.

Применение

  • Ядерная энергетика: используется в ядерных реакторах для поглощения нейтронов.
  • Медицина: применяется в радиоизотопных источниках для лечения рака.
  • Оптика: добавляется в люминофоры для ЖК-экранов (красный цвет).

Интересные факты

  • Лютеций используется в светодиодах для создания красного света.
  • Его оксид Lu₂O₃ добавляют в стекло для защиты от ультрафиолета.
  • Соединения лютеция применяются в лазерах для медицинских процедур.
