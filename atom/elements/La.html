Лантан (La)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 6.16 г/см³
  • Температура плавления: 920 °C
  • Электропроводность: средняя
  • Теплопроводность: средняя

Химические свойства

Лантанид, активно окисляется на воздухе и взаимодействует с водой:

  • С кислородом: 4La + 3O₂ → 2La₂O₃
  • С водой: 2La + 6H₂O → 2La(OH)₃ + 3H₂↑
  • С галогенами: 2La + 3Cl₂ → 2LaCl₃
  • С кислотами: La + 3HCl → LaCl₃ + 3/2H₂↑

История открытия

Открыт Карлом Густавом Мосандером в 1839 году при анализе нитрата церия. Название происходит от греческого "lanthanein", что означает "скрытый", так как элемент был трудно выделен.

Применение

  • Катализаторы: используется в нефтехимии для ускорения реакций.
  • Оптика: добавляется в стекла для объективов камер и лазеров.
  • Аккумуляторы: применяется в никель-металлогидридных батареях.
  • Магниты: часть сплавов для создания мощных магнитов.

Интересные факты

  • Лантан — первый из редкоземельных элементов, открытый в 19 веке.
  • Его оксид La₂O₃ используется в производстве оптических стекол.
  • Соединения лантана применяются в люминофорах для ЖК-экранов.
