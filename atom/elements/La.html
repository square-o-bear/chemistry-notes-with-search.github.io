Цезий (Cs)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 1.87 г/см³
  • Температура плавления: 28.42 °C
  • Электропроводность: высокая
  • Теплопроводность: средняя

Химические свойства

Очень активный металл, легко окисляется и реагирует с водой:

  • С кислородом: 4Cs + O₂ → 2Cs₂O
  • С водой: 2Cs + 2H₂O → 2CsOH + H₂↑
  • С галогенами: 2Cs + Cl₂ → 2CsCl
  • С кислотами: Cs + HCl → CsCl + ½H₂↑

История открытия

Открыт Робертом Бунзеном и Густавом Кирхгофом в 1860 году методом спектроскопии. Название происходит от латинского *caesius* («фиолетовый»), из-за характерного фиолетового свечения в спектре.

Применение

  • Атомные часы: основа стандарта времени (Cs-133).
  • Электроника: фотоэлементы и детекторы.
  • Ядерная энергетика: радиоактивный Cs-137 используется в медицине и исследованиях.

Интересные факты

  • Цезий — единственный металл, который плавится при комнатной температуре.
  • Используется в лазерах для передачи времени GPS-спутников.
  • Радиоактивный изотоп Cs-137 применяется в промышленных измерительных приборах.
