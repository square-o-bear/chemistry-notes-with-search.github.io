Гелий (He)

Физические свойства

  • Агрегатное состояние при н.у.: газ
  • Цвет: бесцветный
  • Запах: без запаха
  • Плотность: 0.1786 g/L (при 0°C)
  • Температура кипения: -268.93°C
  • Растворимость: плохо растворяется в воде

Химические свойства

Гелий — благородный газ, почти полностью инертен. Реагирует только при экстремальных условиях:

  • С литием: 6Li + He → Li_6He (при высоком давлении)
  • С углеродом: C + He → CHe (в квантовых точках)
  • С водородом: H_2 + He → H_2He (при сверхнизких температурах)

История открытия

Гелий был открыт в 1868 году Норманом Локьером и Эдвардом Пикерингом во время наблюдения солнечного затмия. Они заметили неизвестную линию в спектре Солнца, которую позже назвали гелием (от греческого "гелиос" — Солнце). Первый образец гелия на Земле был получен в 1895 году Уильямом Рамзи из минерала цеолита.

Применение

  • Сварка: защитная среда для дуговой сварки (Ar + He → смесь).
  • Заправка воздушных шаров: безопасный газ вместо водорода (He → подъём).
  • Медицина: смесь с кислородом для дайвинга (O_2 + He → снижает риск кессонной болезни).
  • Ядерная энергетика: охлаждение реакторов (He → инертный теплоноситель).

Интересные факты

  • Гелий — второй по распространённости элемент во Вселенной (после водорода).
  • Его нельзя заморозить при нормальном давлении — переходит в жидкий гелий только при 25 атм.
  • Используется в суперпроводниках для охлаждения магнитов (He → жидкий при -268°C).
  • На Земле он редкий — образуется при радиоактивном распаде (U → He).
