Водород (H)

Физические свойства

  • Агрегатное состояние при н.у.: газ
  • Цвет: бесцветный
  • Запах: без запаха
  • Плотность: 0.0899 g/L (при 0°C)
  • Температура кипения: -252.87°C
  • Растворимость: плохо растворяется в воде

Химические свойства

  • Реакция с кислородом: 2H₂ + O₂ → 2H₂O (горение с выделением воды)
  • Реакция с металлами: образует гидриды (например, Ca + H₂ → CaH₂)
  • Реакция с кислотами: взаимодействует с сильными кислотами (например, Zn + 2HCl → ZnCl₂ + H₂↑)
  • Окислительно-восстановительные реакции: действует как восстановитель (CuO + H₂ → Cu + H₂O$)

История открытия

Водород был открыт английским химиком Генри Кавендишем в 1766 году. Он получил его в ходе реакции цинка с соляной кислотой:

Zn + 2HCl → ZnCl₂ + H₂↑

Кавендиш обнаружил, что газ горит с характерным хлопком, и назвал его "горючим воздухом". В 1783 году французский учёный Антуан Лавуазье установил, что этот газ состоит из водорода и кислорода, и дал ему название "водород" (от греческих слов "вода" и "рождающий").

Применение

  • Энергетика: топливо для ракетных двигателей, водородные топливные элементы.
  • Химическая промышленность: производство аммиака (N₂ + 3H₂ → 2NH₃), метанола, бензина.
  • Металлургия: очистка металлов от оксидов (Fe₂O₃ + 3H₂ → 2Fe + 3H₂O).
  • Нефтехимия: гидрогенизация нефтепродуктов.

Интересные факты

  • Водород — самый лёгкий элемент периодической таблицы.
  • Составляет около 75% массы Вселенной.
  • Используется в ядерных реакторах в виде тяжёлой воды (D₂O).
  • Третий по распространённости элемент на Земле (после кислорода и углерода).
