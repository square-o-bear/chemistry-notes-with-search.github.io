Европий (Eu)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 5.26 г/см³
  • Температура плавления: 822 °C
  • Электропроводность: средняя
  • Теплопроводность: средняя

Химические свойства

Лантанид, активно окисляется на воздухе:

  • С кислородом: 4Eu + 3O₂ → 2Eu₂O₃
  • С водой: Eu + 3H₂O → Eu(OH)₃ + 3/2H₂↑
  • С галогенами: 2Eu + 3Cl₂ → 2EuCl₃
  • С кислотами: Eu + 3HCl → EuCl₃ + 3/2H₂↑

История открытия

Открыт Шарлем Эйдельманом в 1890 году при анализе минерала бастнезита. Название происходит от Европы, так как это был первый лантанид, открытый европейцем.

Применение

  • Оптика: используется в люминофорах для ЖК-экранов (зелёный цвет).
  • Магниты: добавляется в неодимовые магниты для улучшения свойств.
  • Ядерная энергетика: применяется в ядерных реакторах для контроля нейтронов.

Интересные факты

  • Европий — единственный лантанид, который имеет радиоактивный изотоп Eu-152.
  • Его оксид Eu₂O₃ используется в красителях для стекла и керамики.
  • Соединения европия добавляются в светодиоды для создания зелёного света.
