Полоний (Po)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 9.2 г/см³
  • Температура плавления: 254 °C
  • Радиоактивность: высокая

Химические свойства

Радиоактивный элемент, активно распадается:

  • С кислородом: 2Po + O₂ → 2PoO
  • С водой: Po + 2H₂O → Po(OH)₂ + H₂↑
  • С азотной кислотой: Po + 4HNO₃ → Po(NO₃)₂ + 2NO₂↑ + 2H₂O
  • С соляной кислотой: Po + 2HCl → PoCl₂ + H₂↑

История открытия

Открыт Марией Кюри в 1898 году. Название дано в честь родины исследовательницы — Польши.

Применение

  • Ядерная энергетика: используется в источниках тепла для космических аппаратов.
  • Промышленность: применяется в антистатических устройствах.
  • Медицина: служит источником радиоактивного излучения.

Интересные факты

  • Полоний — один из самых токсичных элементов на Земле.
  • Его радиоактивность использовалась в убийстве Александра Литвиненко.
  • Самый редкий естественный элемент в земной коре.
