Фтор (F)

Физические свойства

  • Агрегатное состояние: газ
  • Цвет: ярко-зелёный
  • Запах: резкий, раздражающий
  • Плотность: 1.696 g/L
  • Температура кипения: -188°C

Химические свойства

Фтор — самый электроотрицательный элемент. Реагирует со всеми веществами:

  • С водородом: H₂ + F₂ → 2HF↑
  • С металлом: 2Na + F₂ → 2NaF
  • С углеродом: C + 2F₂ → CF₄↑
  • С кислородом: O₂ + 2F₂ → 2OF₂

История открытия

Фтор был открыт Андре-Мари Ампером в 1810 году. Первый чистый фтор получил Генри Мостадтом в 1886 году методом электролиза фтористого водорода.

Применение

  • Пластик: производство Teflon (CF₂).
  • Стоматология: фторирование зубов (NaF).
  • Энергетика: ядерное топливо (UF₆).
  • Химия: производство фреонов.

Интересные факты

  • Фтор — единственный галоген, который не существует в природе в свободном виде.
  • Фторированный питьевой водой снижает риск кариеса.
  • Тефлон — наиболее известный продукт фтора.
