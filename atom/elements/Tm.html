Тулий (Tm)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 9.32 г/см³
  • Температура плавления: 1545 °C
  • Электропроводность: средняя
  • Теплопроводность: средняя

Химические свойства

Лантанид, активно взаимодействует с кислородом и водой:

  • С кислородом: 4Tm + 3O₂ → 2Tm₂O₃
  • С водой: Tm + 3H₂O → Tm(OH)₃ + 3/2H₂↑
  • С галогенами: 2Tm + 3Cl₂ → 2TmCl₃
  • С кислотами: Tm + 3HCl → TmCl₃ + 3/2H₂↑

История открытия

Открыт Карлом Гессом в 1879 году при анализе минерала церита. Название дано в честь Туле — древнего имени Скандинавии.

Применение

  • Оптика: используется в люминофорах для ЖК-экранов (красный цвет).
  • Медицина: применяется в радиоизотопных источниках для лечения рака.
  • Ядерная энергетика: добавляется в ядерные реакторы для поглощения нейтронов.

Интересные факты

  • Тулий используется в светодиодах для создания красного света.
  • Его оксид Tm₂O₃ добавляют в стекло для защиты от ультрафиолета.
  • Соединения тулия применяются в лазерах для медицинских процедур.
