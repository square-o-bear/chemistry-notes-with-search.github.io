Рений (Re)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 21.02 г/см³
  • Температура плавления: 3186 °C
  • Электропроводность: высокая
  • Теплопроводность: средняя

Химические свойства

Переходный металл, устойчив к коррозии:

  • С кислородом: 2Re + O₂ → 2ReO
  • С водой: Re + 2H₂O → ReO₂ + 2H₂↑
  • С галогенами: Re + 3F₂ → ReF₆
  • С кислотами: Re + 6HNO₃ → HReO₄ + 6NO₂↑ + 2H₂O

История открытия

Открыт Иоганном Винклером в 1925 году при анализе минерала платины. Название происходит от латинского *Rhenus* — Рейн.

Применение

  • Энергетика: используется в термоядерных реакторах.
  • Авиация: добавляется в сплавы для авиационных двигателей.
  • Катализаторы: применяются в нефтепереработке.

Интересные факты

  • Рений имеет вторую самую высокую температуру плавления среди всех металлов.
  • Его оксид ReO₃ используется в электрокристаллических стеклах.
  • Соединения рения применяются в катализаторах для синтеза аммиака.
