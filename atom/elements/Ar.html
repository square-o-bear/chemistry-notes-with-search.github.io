Аргон (Ar)

Физические свойства

  • Агрегатное состояние: газ
  • Цвет: бесцветный
  • Запах: без запаха
  • Плотность: 1.784 g/L
  • Температура кипения: -185.8°C
  • Растворимость: плохо растворяется в воде

Химические свойства

Аргон — инертный газ, не образует соединений при обычных условиях:

  • Не реагирует с другими элементами.
  • Может взаимодействовать только при сверхвысоком давлении и температуре.

История открытия

Аргон был открыт Уильямом Рамзи и Морисом Траверсом в 1894 году при разделении воздуха. Название происходит от греческого "argos" — ленивый.

Применение

  • Сварка: защитная атмосфера (Ar → предотвращает окисление).
  • Светотехника: люминесцентные лампы (Ar → улучшает свечение).
  • Наука: заполнение баллонов для экспериментов.

Интересные факты

  • Аргон составляет около 0.93% атмосферы Земли.
  • Он используется в лазерах для медицинских процедур.
  • Аргон — один из самых распространённых инертных газов.
