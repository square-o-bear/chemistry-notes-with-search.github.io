Осмий (Os)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 22.59 г/см³
  • Температура плавления: 3033 °C
  • Электропроводность: высокая
  • Теплопроводность: средняя

Химические свойства

Переходный металл, устойчив к коррозии:

  • С кислородом: 2Os + O₂ → 2OsO
  • С водой: Os + 2H₂O → OsO₂ + 2H₂↑
  • С галогенами: Os + 3F₂ → OsF₆
  • С кислотами: Os + 6HNO₃ → H₂OsO₄ + 6NO₂↑ + 2H₂O

История открытия

Открыт Смитсоном Теннантом в 1803 году при анализе платинового остатка. Название происходит от греческого *osme* («запах»), из-за запаха его оксида.

Применение

  • Энергетика: используется в термоядерных реакторах.
  • Авиация: добавляется в сплавы для авиационных двигателей.
  • Катализаторы: применяются в нефтепереработке.

Интересные факты

  • Осмий — самый плотный естественный элемент на Земле.
  • Его оксид OsO₄ используется в микроскопии для фиксации тканей.
  • Соединения осмия применяются в катализаторах для синтеза аммиака.
