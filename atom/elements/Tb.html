Тербий (Tb)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 8.23 г/см³
  • Температура плавления: 1356 °C
  • Электропроводность: низкая
  • Теплопроводность: средняя

Химические свойства

Лантанид, активно окисляется на воздухе и взаимодействует с водой:

  • С кислородом: 4Tb + 3O₂ → 2Tb₂O₃
  • С водой: 2Tb + 6H₂O → 2Tb(OH)₃ + 3H₂↑
  • С галогенами: 2Tb + 3Cl₂ → 2TbCl₃
  • С кислотами: Tb + 3HCl → TbCl₃ + 3/2H₂↑

История открытия

Открыт Карлом Густавом Мосандером в 1843 году при анализе дидима. Название происходит от "Terbium", латинского имени богини Тервы из скандинавской мифологии.

Применение

  • Магниты: используется в неодимово-тербиевых магнитах для улучшения термостойкости.
  • Оптика: добавляется в люминофоры для ЖК-экранов (зелёный свет).
  • Ядерная энергетика: Tb₂O₃ применяется в поглотителях нейтронов.
  • Медицина: соединения тербия используются в диагностических радиофармацевтиках.

Интересные факты

  • Тербий обладает высокой способностью к люминесценции под воздействием ультрафиолетового света.
  • Его оксид Tb₂O₃ используется в защитных стеклах от радиации.
  • Соединения тербия добавляются в светодиоды для создания яркого зелёного цвета.
