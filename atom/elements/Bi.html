Висмут (Bi)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: синевато-серый
  • Плотность: 9.78 г/см³
  • Температура плавления: 271 °C
  • Электропроводность: низкая
  • Теплопроводность: низкая

Химические свойства

Тяжёлый металл, образует оксидную плёнку:

  • С кислородом: 2Bi + O₂ → 2BiO
  • С водой: Bi + H₂O → Bi(OH)₃ + H₂↑
  • С азотной кислотой: Bi + 4HNO₃ → Bi(NO₃)₃ + NO↑ + 2H₂O
  • С соляной кислотой: Bi + 2HCl → BiCl₃ + H₂↑

История открытия

Открыт Георгиус Агрекола в 1540 году. Название происходит от немецкого *Wismut* («белый свинец»).

Применение

  • Медицина: используется в лекарствах при язве желудка.
  • Промышленность: добавляется в сплавы для снижения температуры плавки.
  • Электроника: применяется в термоэлектрических устройствах.

Интересные факты

  • Висмут обладает самой низкой среди металлов температурой плавления.
  • Его кристаллы имеют ромбическую форму.
  • Считается безопасным для здоровья, несмотря на радиоактивность.
