Тантал (Ta)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 16.65 г/см³
  • Температура плавления: 3017 °C
  • Электропроводность: средняя
  • Теплопроводность: средняя

Химические свойства

Переходный металл, почти не реагирует с кислотами:

  • С кислородом: 2Ta + O₂ → 2TaO
  • С водой: Ta + 2H₂O → TaO₂ + 2H₂↑
  • С галогенами: Ta + 3F₂ → TaF₆
  • С кислотами: Ta + 6HF → TaF₆ + 3H₂↑

История открытия

Открыт Андерсом Эгестадом в 1802 году при анализе минерала колумбита. Название дано в честь титана Танталия из греческой мифологии.

Применение

  • Электроника: используется в конденсаторах для повышения ёмкости.
  • Медицина: применяется в хирургических инструментах и имплантах.
  • Авиация: добавляется в сплавы для авиационных двигателей.

Интересные факты

  • Тантал почти не подвержен коррозии даже в агрессивных средах.
  • Его оксид Ta₂O₅ используется в оптических стеклах.
  • Соединения тантала применяются в катализаторах для нефтепереработки.
