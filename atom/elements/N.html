Азот (N)

Физические свойства

  • Агрегатное состояние при н.у.: газ
  • Цвет: бесцветный
  • Запах: без запаха
  • Плотность: 1.25 g/L (при 0°C)
  • Температура кипения: -195.8°C
  • Растворимость: плохо растворяется в воде

Химические свойства

Азот — двухатомный газ, относительно инертен при комнатной температуре. Реагирует при высоких температурах или под действием электрического разряда:

  • С водородом: N₂ + 3H₂ → 2NH₃↑ (синтез аммиака)
  • С кислородом: N₂ + O₂ → 2NO↑ (при сварке или молнии)
  • С литием: 6Li + N₂ → 2Li₃N
  • С магнием: 3Mg + N₂ → Mg₃N₂

История открытия

Азот был открыт в 1772 году шотландским химиком Дэниелом Ранделлом. Он обнаружил, что воздух содержит газ, который не поддерживает горение и не дыхание. Впоследствии Карл Вильгельм Шееле и Джозеф Пристли также исследовали этот газ. Название "азот" происходит от греческого "a-" (без) и "zoe" (жизнь), так как он не поддерживает жизнь.

Применение

  • Азотные удобрения: производство аммиака (N₂ → NH₃).
  • Медицина: жидкий азот для криохирургии (N₂ → охлаждение).
  • Промышленность: защитная атмосфера при пайке (N₂ → предотвращает окисление).
  • Электроника: очистка оборудования (N₂ → инертный газ).
  • Пищевая промышленность: упаковка продуктов (N₂ → продлевает срок годности).

Интересные факты

  • Азот составляет 78% атмосферы Земли.
  • Жидкий азот используется для быстрого замораживания.
  • Без азота невозможны белки и ДНК.
  • Азотная кислота (HNO₃) — важное вещество в производстве взрывчатых веществ.
