Самарий (Sm)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 7.52 г/см³
  • Температура плавления: 1072 °C
  • Электропроводность: средняя
  • Теплопроводность: средняя

Химические свойства

Лантанид, активно взаимодействует с кислородом и водой:

  • С кислородом: 4Sm + 3O₂ → 2Sm₂O₃
  • С водой: Sm + 3H₂O → Sm(OH)₃ + 3/2H₂↑
  • С галогенами: 2Sm + 3Cl₂ → 2SmCl₃
  • С кислотами: Sm + 3HCl → SmCl₃ + 3/2H₂↑

История открытия

Открыт Луи Эйдельманом в 1879 году при анализе минерала самарскита. Название дано в честь русского министра Самарина.

Применение

  • Магниты: используется в постоянных магнитах для двигателей.
  • Ядерная энергетика: добавляется в ядерные реакторы для контроля реакций.
  • Оптика: применяется в лазерах и люминофорах.

Интересные факты

  • Самарий используется в светодиодах для создания синего света.
  • Его оксид Sm₂O₃ добавляют в стекло для защиты от ультрафиолета.
  • Самарий был первым лантанидом, открытый в России.
