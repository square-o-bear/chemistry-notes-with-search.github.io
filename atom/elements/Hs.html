Хассий (Hs)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: неизвестен
  • Плотность: 41 г/см³ (оценка)
  • Температура плавления: неизвестна
  • Температура кипения: неизвестна
  • Радиоактивность: все изотопы нестабильны

Химические свойства

Хассий - переходный металл, 8-я группа. По свойствам близок к осмию и рутению. Степени окисления +8, +6, +4.

  • С кислородом: Hs + 4O₂ → HsO₄
  • С галогенами: Hs + 4F₂ → HsF₈
  • С водой: предположительно устойчив
  • С кислотами: предположительно растворяется

История открытия

Открыт в 1984 году в Центре исследований тяжёлых ионов (GSI, Германия). Назван в честь немецкой земли Гессен (Hassia), где расположен Дармштадт.

Применение

  • Научные исследования: изучение химии 8-й группы тяжёлых элементов.
  • Ядерная физика: исследование свойств сверхтяжёлых ядер.
  • Химия: проверка периодических закономерностей.

Интересные факты

  • Хассий - один из самых тяжёлых элементов с изученными химическими свойствами.
  • Период полураспада ²⁶⁹Hs - 9,7 секунды (наиболее стабильный изотоп).
  • Образует летучий тетраоксид HsO₄, аналогично OsO₄.
  • Получается облучением свинца ионами железа.
