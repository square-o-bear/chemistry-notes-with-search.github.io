Таллий (Tl)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 11.85 г/см³
  • Температура плавления: 304 °C
  • Электропроводность: высокая
  • Теплопроводность: средняя

Химические свойства

Полуметалл, легко окисляется:

  • С кислородом: 2Tl + O₂ → 2TlO
  • С водой: Tl + H₂O → TlOH + H₂↑
  • С азотной кислотой: Tl + 4HNO₃ → Tl(NO₃)₃ + NO↑ + 2H₂O
  • С соляной кислотой: Tl + 2HCl → TlCl₂ + H₂↑

История открытия

Открыт Уильямом Кроуфордом в 1861 году с помощью спектроскопии. Название происходит от греческого *thallos* («зелёный побег»), из-за зелёного свечения его спектра.

Применение

  • Электроника: используется в фотоэлементах и диодах.
  • Медицина: применяется в радиофармацевтике для диагностики.
  • Промышленность: добавляется в светодиоды для улучшения светоотдачи.

Интересные факты

  • Таллий токсичен и был использован в нескольких известных случаях убийств.
  • Его соединения применяются в оптических стеклах.
  • Считается одним из самых опасных металлов.
