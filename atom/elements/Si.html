Кремний (Si)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серый или чёрный
  • Плотность: 2.33 g/cm³
  • Температура плавления: 1414°C
  • Электропроводность: высокая (полупроводник)

Химические свойства

Кремний — амфотерный элемент, реагирует с кислотами и щелочами:

  • С кислородом: Si + O₂ → SiO₂
  • С водородом: Si + 2H₂ → SiH₄↑
  • С хлором: Si + 2Cl₂ → SiCl₄
  • С щелочью: Si + 2NaOH + H₂O → Na₂SiO₃ + 2H₂↑

История открытия

Кремний был открыт Джозефом Листером в 1823 году при анализе кремнезёма. Его название происходит от латинского "silex" — камень.

Применение

  • Электроника: полупроводники (Si → чипы, транзисторы).
  • Стекло: кремнезём (SiO₂ → стекло, керамика).
  • Энергетика: солнечные батареи (Si → фотоэлементы).

Интересные факты

  • Кремний — второй по распространённости элемент в земной коре.
  • Он используется в производстве герметиков и силиконов.
  • Без кремния невозможны современные компьютеры.
