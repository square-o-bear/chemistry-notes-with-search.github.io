Олово (Sn)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 7.31 г/см³ (белое), 5.77 г/см³ (серое)
  • Температура плавления: 231.9°C
  • Температура кипения: 2602°C
  • Аллотропия: белое олово (металлическое) и серое олово (полупроводник)

Химические свойства

Олово устойчиво к коррозии, образует защитную плёнку SnO₂.

  • С кислородом: Sn + O₂ → SnO₂
  • С кислотами: Sn + 2HCl → SnCl₂ + H₂↑
  • С щелочами: Sn + 2NaOH + 2H₂O → Na₂[Sn(OH)₄] + H₂↑
  • С галогенами: Sn + 2Cl₂ → SnCl₄

История открытия

Известно с древности (бронзовый век). Название от латинского stannum. Символ Sn — от него же. Использовалось в сплавах с медью — бронза.

Применение

  • Лужёная жесть: оловянное покрытие стали для консервных банок.
  • Припои: сплавы Sn-Pb и безсвинцовые (Sn-Ag-Cu).
  • Сплавы: бронза, пьютер, баббит.
  • Химическая промышленность: катализаторы (SnCl₂).

Интересные факты

  • При сгибании олово издаёт "оловянный крик" — скрип кристаллов.
  • При температуре ниже 13°C превращается в серое олово — "оловянная чума".
  • Олово безопасно — используется в детских игрушках и посуде.
  • Безсвинцовые припои — важны для экологии электроники.
