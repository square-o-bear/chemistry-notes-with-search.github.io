Цинк (Zn)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 7.14 g/cm³
  • Температура плавления: 419.5°C
  • Электропроводность: средняя

Химические свойства

Цинк — амфотерный металл, легко реагирует с кислотами:

  • С кислородом: 2Zn + O₂ → 2ZnO
  • С соляной кислотой: Zn + 2HCl → ZnCl₂ + H₂↑
  • С щелочью: Zn + 2NaOH + H₂O → Na₂ZnO₂ + 2H₂↑

История открытия

Цинк был выделен Андреасом Маргграфом в 1746 году при перегонке ртути. Название происходит от немецкого "Zinke" — остриё.

Применение

  • Гальваника: защита железа от коррозии (Zn → покрытие).
  • Медицина: антисептики (ZnO → лечение кожи).
  • Производство: лампочки (ZnS → люминофор).

Интересные факты

  • Цинк необходим для иммунитета и роста организма.
  • Он используется в производстве батареек.
  • Цинк — один из важнейших микроэлементов в организме.
