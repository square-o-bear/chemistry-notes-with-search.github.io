Бор (B)

Физические свойства

  • Агрегатное состояние при н.у.: твёрдое вещество
  • Цвет: чёрно-коричневый
  • Плотность: 2.34 g/cm³
  • Температура плавления: 2076°C
  • Электропроводность: зависит от формы (полупроводник)

Химические свойства

Бор — амфотерный элемент, реагирует с кислотами и щелочами:

  • С кислородом: 4B + 3O₂ → 2B₂O₃
  • С хлором: 2B + 3Cl₂ → 2BCl₃
  • С водой: не реагирует при комнатной температуре
  • С щелочами: 2B + 2NaOH + 3H₂O → 2NaBO₂ + 3H₂↑

История открытия

Бор был открыт в 1808 году французскими химиками Гийомом Гей-Люссаком и Луи Жаном Тенаром. Они получили его из минерала боракса, нагревая его с металлическим калием:

K₂B₄O₇ + 4K → 4B + K₂O

Название "бор" происходит от арабского слова "бура", которое описывает минерал боракс.

Применение

  • Стекло и керамика: добавка для повышения термостойкости (B₂O₃).
  • Полупроводники: легирование германия и кремния (B → примесь).
  • Ядерная энергетика: поглотитель нейтронов (B → контроль реакции).
  • Медицина: антисептики и мази (B → компонент борной кислоты).

Интересные факты

  • Бор имеет 10 известных аллотропных модификаций.
  • Наиболее распространённая форма — ромбическая.
  • Борная кислота используется в очистке глаз.
  • Бор — единственный неметалл, который может проводить электричество.
