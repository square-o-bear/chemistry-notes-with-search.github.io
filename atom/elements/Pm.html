Прометий (Pm)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 7.26 г/см³
  • Температура плавления: 1100 °C
  • Электропроводность: средняя
  • Теплопроводность: средняя

Химические свойства

Радиоактивный лантанид, не существует в природе:

  • С кислородом: 4Pm + 3O₂ → 2Pm₂O₃
  • С водой: Pm + 3H₂O → Pm(OH)₃ + 3/2H₂↑
  • С галогенами: 2Pm + 3Cl₂ → 2PmCl₃
  • С кислотами: Pm + 3HCl → PmCl₃ + 3/2H₂↑

История открытия

Открыт Чарльзом Дикманом и Лайонелом Хьюиттом в 1945 году при ядерных исследованиях. Название происходит от греческого *Prometheus* («пророчество»), как символ опасности радиоактивности.

Применение

  • Ядерная энергетика: используется в атомных батареях.
  • Медицина: применяется в радиоизотопной терапии.
  • Научные исследования: изучается для создания новых материалов.

Интересные факты

  • Прометий — единственный лантанид, полностью радиоактивный.
  • Его изотоп Pm-147 используется в освещении космических аппаратов.
  • Соединения прометия добавляются в люминофоры для синего света.
