Эрбий (Er)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 9.07 г/см³
  • Температура плавления: 1497 °C
  • Электропроводность: средняя
  • Теплопроводность: средняя

Химические свойства

Лантанид, активно реагирует с кислородом и водой:

  • С кислородом: 4Er + 3O₂ → 2Er₂O₃
  • С водой: Er + 3H₂O → Er(OH)₃ + 3/2H₂↑
  • С галогенами: 2Er + 3Cl₂ → 2ErCl₃
  • С кислотами: Er + 3HCl → ErCl₃ + 3/2H₂↑

История открытия

Открыт Шарлем Эйдельманом в 1843 году при анализе минерала эльсниевита. Название дано в честь горы Якобсберга в Швеции.

Применение

  • Оптика: используется в люминофорах для ЖК-экранов (красный цвет).
  • Медицина: применяется в лазерах для хирургических операций.
  • Ядерная энергетика: добавляется в ядерные реакторы для контроля нейтронов.

Интересные факты

  • Эрбий используется в светодиодах для создания красного света.
  • Его оксид Er₂O₃ добавляют в стекло для защиты от ультрафиолета.
  • Соединения эрбия применяются в лазерах для медицинских процедур.
