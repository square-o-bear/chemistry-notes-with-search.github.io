Платина (Pt)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 21.45 г/см³
  • Температура плавления: 1768 °C
  • Электропроводность: высокая
  • Теплопроводность: средняя

Химические свойства

Переходный металл, устойчив к коррозии:

  • С кислородом: 2Pt + O₂ → 2PtO
  • С водой: Pt + 2H₂O → PtO₂ + 2H₂↑
  • С галогенами: Pt + 2F₂ → PtF₄
  • С кислотами: Pt + 6HNO₃ → H₂PtCl₆ + 6NO₂↑ + 2H₂O

История открытия

Открыт Антонио де Ульяро в 1735 году при анализе минерала платинита. Название происходит от испанского *platina* («маленькая серебряная руда»).

Применение

  • Энергетика: используется в термоядерных реакторах.
  • Авиация: добавляется в сплавы для авиационных двигателей.
  • Катализаторы: применяются в нефтепереработке.

Интересные факты

  • Платина используется в ювелирных изделиях за её устойчивость к коррозии.
  • Её оксид PtO₂ используется в электролитических конденсаторах.
  • Соединения платины применяются в катализаторах для синтеза аммиака.
