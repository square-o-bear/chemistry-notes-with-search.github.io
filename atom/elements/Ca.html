Кальций (Ca)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 1.55 g/cm³
  • Температура плавления: 842°C
  • Электропроводность: средняя

Химические свойства

Кальций — активный щелочноземельный металл, образует оксид при нагревании:

  • С водой: Ca + 2H₂O → Ca(OH)₂ + H₂↑
  • С хлором: Ca + Cl₂ → CaCl₂
  • С кислородом: 2Ca + O₂ → 2CaO

История открытия

Кальций был открыт Гемфри Дэви в 1808 году методом электролиза оксида кальция.

Применение

  • Строительство: цемент и бетон (CaCO₃ → основа материалов).
  • Медицина: кости и зубы (Ca → структурная основа).
  • Промышленность: производство стали (Ca → удаление примесей).

Интересные факты

  • Кальций — третий по распространённости элемент в организме человека.
  • Он используется в производстве светодиодов.
  • Кальций необходим для коагуляции крови.
