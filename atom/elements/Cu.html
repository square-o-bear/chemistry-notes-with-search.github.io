Медь (Cu)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: красно-золотистый
  • Плотность: 8.96 g/cm³
  • Температура плавления: 1085°C
  • Электропроводность: очень высокая

Химические свойства

Медь — активный металл, окисляется на воздухе:

  • С кислородом: 2Cu + O₂ → 2CuO
  • С хлором: Cu + Cl₂ → CuCl₂
  • С азотной кислотой: Cu + 4HNO₃ → Cu(NO₃)₂ + 2NO₂↑ + 2H₂O

История открытия

Медь была известна ещё в древности. Современный элемент был описан Аристотелем в IV веке до н. э.

Применение

  • Электротехника: провода (Cu → высокая проводимость).
  • Строительство: кровли и трубы (Cu → антикоррозионность).
  • Монеты: сплавы с цинком (Cu-Zn → мельхиор).

Интересные факты

  • Медь — один из немногих металлов, которые были использованы в доисторические времена.
  • Она убивает бактерии (антибактериальные свойства).
  • Медь необходима для работы организма (ферменты, кровь).
