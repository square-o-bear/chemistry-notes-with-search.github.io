Селен (Se)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: красный или чёрный
  • Плотность: 4.81 g/cm³
  • Температура плавления: 221°C
  • Электропроводность: зависит от формы

Химические свойства

Селен — неметалл, образует оксиды и соли:

  • С кислородом: Se + O₂ → SeO₂
  • С водородом: Se + H₂ → H₂Se↑
  • С металлами: Se + 2Na → Na₂Se

История открытия

Селен был открыт Йёнсом Берцелиусом в 1817 году при анализе серы. Название происходит от древнегреческой богини Селены — луны.

Применение

  • Электроника: фотоэлементы (Se → преобразование света в электричество).
  • Стекло: удаление цвета (Se → нейтрализация зелёного оттенка).
  • Биология: антиоксидант (Se → защита клеток от окисления).

Интересные факты

  • Селен необходим для работы щитовидной железы.
  • Он используется в производстве красителей.
  • Селен токсичен в больших количествах.
