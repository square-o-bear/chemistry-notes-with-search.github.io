Теннессин (Ts)

Физические свойства

  • Агрегатное состояние: предположительно твёрдое
  • Цвет: неизвестен
  • Плотность: 7,2 г/см³ (оценка)
  • Температура плавления: 450°C (оценка)
  • Температура кипения: 610°C (оценка)
  • Радиоактивность: все изотопы нестабильны

Химические свойства

Теннессин - галоген, 17-я группа. По свойствам близок к астату, йоду и брому. Степени окисления +7, +5, +3, +1, -1.

  • С кислородом: 2Ts + 7O₂ → 2Ts₂O₇
  • С водородом: Ts + H₂ → 2HTs
  • С металлами: 2Na + Ts₂ → 2NaTs
  • С водой: предположительно реагирует

История открытия

Открыт в 2010 году в ОИЯИ (Дубна, Россия) совместно с США. Назван в честь штата Теннесси, где расположены исследовательские лаборатории.

Применение

  • Научные исследования: изучение химии галогенов.
  • Ядерная физика: исследование свойств сверхтяжёлых ядер.
  • Химия: проверка периодических закономерностей.

Интересные факты

  • Теннессин - второй галоген после астата, не имеющий стабильных изотопов.
  • Период полураспада ²⁹⁴Ts - 51 миллисекунда (наиболее стабильный изотоп).
  • Может проявлять металлические свойства из-за релятивистских эффектов.
  • Получается облучением берклия ионами кальция.
