Марганец (Mn)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 7.21 g/cm³
  • Температура плавления: 1246°C
  • Электропроводность: низкая

Химические свойства

Марганец — амфотерный металл, проявляет разнообразные степени окисления:

  • С кислородом: 2Mn + O₂ → 2MnO
  • С хлором: Mn + Cl₂ → MnCl₂
  • С водой: Mn + 2H₂O → Mn(OH)₂ + H₂↑

История открытия

Марганец был открыт Йоганном Готтлибом Ганом в 1774 году при анализе миндаля. Название происходит от древнегреческого "manganês" — миндальное дерево.

Применение

  • Сталь: легирование (Mn → повышает прочность).
  • Батареи: сухие элементы (MnO₂ → катодный материал).
  • Сельское хозяйство: удобрения (Mn → микроэлемент для растений).

Интересные факты

  • Марганец необходим для синтеза белка в организме.
  • Он используется в производстве фиолетовых стекол.
  • Марганец — один из важнейших элементов в металлургии.
