Криптон (Kr)

Физические свойства

  • Агрегатное состояние: газ
  • Цвет: бесцветный
  • Плотность: 3.74 g/L
  • Температура кипения: -153.4°C
  • Растворимость: плохо растворяется в воде

Химические свойства

Криптон — инертный газ, не образует соединений при обычных условиях:

  • Не реагирует с другими элементами.
  • Может взаимодействовать только при сверхвысоком давлении и температуре.

История открытия

Криптон был открыт Уильямом Рамзи и Морисом Траверсом в 1898 году при разделении воздуха. Название происходит от греческого "kryptos" — скрытый.

Применение

  • Светотехника: люминесцентные лампы (Kr → улучшает свечение).
  • Наука: заполнение баллонов для экспериментов.
  • Энергетика: теплоизоляция (Kr → энергоэффективные окна).

Интересные факты

  • Криптон составляет около 1.14 ppm атмосферы Земли.
  • Он используется в лазерах для медицинских процедур.
  • Криптон — один из самых распространённых инертных газов.
