Диспрозий (Dy)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 8.55 г/см³
  • Температура плавления: 1412 °C
  • Электропроводность: низкая
  • Теплопроводность: средняя

Химические свойства

Лантанид, активно окисляется на воздухе и взаимодействует с водой:

  • С кислородом: 4Dy + 3O₂ → 2Dy₂O₃
  • С водой: 2Dy + 6H₂O → 2Dy(OH)₃ + 3H₂↑
  • С галогенами: 2Dy + 3Cl₂ → 2DyCl₃
  • С кислотами: Dy + 3HCl → DyCl₃ + 3/2H₂↑

История открытия

Открыт Полем Эмилем Лекок де Буабодраном в 1886 году при анализе оксида празеодима. Название происходит от греческого "dysprositos", что означает "труднодоступный".

Применение

  • Магниты: используется в неодимово-диспрозиевых магнитах для улучшения термостойкости.
  • Ядерная энергетика: Dy₂O₃ применяется в поглотителях нейтронов.
  • Оптика: добавляется в люминофоры для светодиодов (красный свет).
  • Электроника: используется в лазерах и стабилизаторах напряжения.

Интересные факты

  • Диспрозий обладает одним из самых высоких магнитных моментов среди всех элементов.
  • Его оксид Dy₂O₃ используется в защитных стеклах от радиации.
  • Соединения диспрозия добавляются в светодиоды для создания яркого красного цвета.
