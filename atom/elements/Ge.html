Германий (Ge)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серый
  • Плотность: 5.32 g/cm³
  • Температура плавления: 938.25°C
  • Электропроводность: высокая (полупроводник)

Химические свойства

Германий — амфотерный элемент, проявляет разнообразные степени окисления:

  • С кислородом: Ge + O₂ → GeO₂
  • С водородом: Ge + 2H₂ → GeH₄↑
  • С хлором: Ge + 2Cl₂ → GeCl₄

История открытия

Германий был предсказан Дмитрием Менделеевым как "эка-кремний". Открыт Клеменс Винклером в 1886 году в арсенопирите.

Применение

  • Электроника: полупроводники (Ge → транзисторы, фотоэлементы).
  • Оптика: линзы и фильтры (GeO₂ → инфракрасные материалы).
  • Энергетика: солнечные батареи (Ge → подложки для Si).

Интересные факты

  • Германий используется в производстве сверхпроводящих магнитов.
  • Он применяется в системах ночного видения.
  • Германий — один из немногих элементов, предсказанных Менделеевым.
