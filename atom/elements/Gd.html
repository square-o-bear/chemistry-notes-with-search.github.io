Гадолиний (Gd)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 7.895 г/см³
  • Температура плавления: 1313 °C
  • Электропроводность: низкая
  • Теплопроводность: средняя

Химические свойства

Лантанид, активно взаимодействует с кислородом и водой:

  • С кислородом: 4Gd + 3O₂ → 2Gd₂O₃
  • С водой: 2Gd + 6H₂O → 2Gd(OH)₃ + 3H₂↑
  • С галогенами: 2Gd + 3Cl₂ → 2GdCl₃
  • С кислотами: Gd + 3HCl → GdCl₃ + 3/2H₂↑

История открытия

Открыт Жаном Галисард де Маригнаком в 1880 году при анализе оксида иттрия (Y₂O₃). Название дано в честь шведского учёного Кристиана Гадолина.

Применение

  • Ядерная энергетика: используется в поглотителях нейтронов для контроля реакций деления.
  • Медицина: Gd₂O₃ применяется в магнитно-резонансной томографии (контрастные агенты).
  • Оптика: добавляется в люминофоры для светодиодов (красный свет).
  • Сплавы: улучшает магнитные свойства в редкоземельных магнитах.

Интересные факты

  • Гадолиний имеет один из самых высоких коэффициентов захвата тепловых нейтронов (Gd-157).
  • Его оксид Gd₂O₃ используется в стеклах для защиты от радиации.
  • Гадолиний демонстрирует сильный парамагнетизм при низких температурах.
