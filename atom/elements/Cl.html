Хлор (Cl)

Физические свойства

  • Агрегатное состояние: газ
  • Цвет: зелёно-жёлтый
  • Запах: резкий, раздражающий
  • Плотность: 3.2 g/L
  • Температура кипения: -34.04°C

Химические свойства

Хлор — активный галоген, легко реагирует с металлами и водородом:

  • С металлом: 2Na + Cl₂ → 2NaCl
  • С водородом: H₂ + Cl₂ → 2HCl↑
  • С водой: Cl₂ + H₂O → HCl + HOCl

История открытия

Хлор был открыт Карлом Вильгельмом Шееле в 1774 году. Название происходит от греческого "chloros" — зелёный.

Применение

  • Обеззараживание: хлорирование воды (Cl₂ → уничтожение бактерий).
  • Химия: производство пластмасс (Cl → ПВХ).
  • Медицина: антисептики (Cl → хлориды).

Интересные факты

  • Хлор — третий по электроотрицательности элемент после фтора и кислорода.
  • Он используется в производстве хлорной извести.
  • Хлор необходим для работы желудка (HCl → желудочный сок).
