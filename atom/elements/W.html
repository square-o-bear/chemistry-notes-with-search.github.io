Вольфрам (W)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 19.3 г/см³
  • Температура плавления: 3422 °C
  • Электропроводность: высокая
  • Теплопроводность: высокая

Химические свойства

Переходный металл, устойчив к коррозии:

  • С кислородом: W + O₂ → WO₂
  • С водой: W + 2H₂O → WO₂ + 2H₂↑
  • С галогенами: W + 3F₂ → WF₆
  • С кислотами: W + 6HNO₃ → H₂WO₄ + 6NO₂↑ + 2H₂O

История открытия

Открыт Карлосом Ириарете и Фернандо дель Рио в 1781 году при анализе минерала вольфрамита. Название происходит от немецкого *Wolfram* — «волчья борода».

Применение

  • Электроника: используется в нитях ламп накаливания.
  • Авиация: добавляется в сплавы для авиационных двигателей.
  • Энергетика: применяется в термоядерных реакторах.

Интересные факты

  • Вольфрам имеет самую высокую температуру плавления среди всех металлов.
  • Его оксид WO₃ используется в электрокристаллических стеклах.
  • Соединения вольфрама применяются в катализаторах для нефтепереработки.
