Празеодим (Pr)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 6.77 г/см³
  • Температура плавления: 935 °C
  • Электропроводность: средняя
  • Теплопроводность: средняя

Химические свойства

Лантанид, легко окисляется на воздухе:

  • С кислородом: 4Pr + 3O₂ → 2Pr₂O₃
  • С водой: Pr + 3H₂O → Pr(OH)₃ + 3/2H₂↑
  • С галогенами: 2Pr + 3Cl₂ → 2PrCl₃
  • С кислотами: Pr + 3HCl → PrCl₃ + 3/2H₂↑

История открытия

Открыт Карлом Гессом и Лайнисом Фитцером в 1841 году при анализе минерала церита. Название происходит от греческого *praseus* («зелёный»), из-за зелёного цвета его соединений.

Применение

  • Магниты: используется в неодимовых магнитах для улучшения прочности.
  • Оптика: добавляется в стекло для фильтрации инфракрасного света.
  • Катализаторы: применяется в нефтепереработке.

Интересные факты

  • Празеодимовые сплавы используются в лазерах.
  • Его оксид Pr₂O₃ добавляют в стекло для уменьшения желтого оттенка.
  • Соединения празеодима используются в люминофорах для ЖК-экранов.
