Гафний (Hf)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: серебристо-белый
  • Плотность: 13.31 г/см³
  • Температура плавления: 2233 °C
  • Электропроводность: средняя
  • Теплопроводность: средняя

Химические свойства

Переходный металл, образует прочные соединения с кислородом:

  • С кислородом: 2Hf + O₂ → 2HfO
  • С водой: Hf + 2H₂O → HfO₂ + 2H₂↑
  • С галогенами: Hf + 2F₂ → HfF₄
  • С кислотами: Hf + 4HNO₃ → Hf(NO₃)₄ + 2H₂↑

История открытия

Открыт Дагом Дельсоном и Торбергом Магнусом в 1923 году при анализе минерала циркония. Название происходит от латинского *Hafnia* — Копенгаген.

Применение

  • Ядерная энергетика: используется в ядерных реакторах как поглотитель нейтронов.
  • Авиация: добавляется в сплавы для авиационных двигателей.
  • Электроника: применяется в микросхемах для улучшения проводимости.

Интересные факты

  • Гафний обладает высокой стойкостью к коррозии.
  • Его оксид HfO₂ используется в изоляционных слоях полупроводников.
  • Соединения гафния применяются в катализаторах для нефтепереработки.
