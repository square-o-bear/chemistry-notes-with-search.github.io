Рентгений (Rg)

Физические свойства

  • Агрегатное состояние: твёрдое
  • Цвет: неизвестен
  • Плотность: 28,7 г/см³ (оценка)
  • Температура плавления: неизвестна
  • Температура кипения: неизвестна
  • Радиоактивность: все изотопы нестабильны

Химические свойства

Рентгений - переходный металл, 11-я группа. По свойствам близок к золоту, серебру и меди. Степени окисления +5, +3, +1.

  • С кислородом: 2Rg + O₂ → 2Rg₂O
  • С галогенами: 2Rg + Cl₂ → 2RgCl
  • С водой: предположительно устойчив
  • С кислотами: предположительно растворяется

История открытия

Открыт в 1994 году в Центре исследований тяжёлых ионов (GSI, Германия). Назван в честь Вильгельма Конрада Рентгена, открывателя рентгеновских лучей.

Применение

  • Научные исследования: изучение химии 11-й группы тяжёлых элементов.
  • Ядерная физика: исследование свойств сверхтяжёлых ядер.
  • Химия: проверка периодических закономерностей.

Интересные факты

  • Рентген получил первую Нобелевскую премию по физике (1901).
  • Период полураспада ²⁸²Rg - 2,1 минуты (наиболее стабильный изотоп).
  • Предположительно может быть благородным металлом, подобно золоту.
  • Получается облучением висмута ионами никеля.
