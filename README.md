О чём это репозиторий?
=====================
Этот репозиторий является конспектом по химии. 
[Вот сайтик потыкаться](https://square-o-bear.github.io/chemistry-notes-with-search.github.io/index.html)

Про поиск
=====================
1. Введите интересующее вас слово/главу/информацию, и система выдаст список страниц с ключевыми словами, подходящими под тему
2. Все страницы имеют свои ключевые слова с их весами. Чем больше их суммарный вес
3. Строка поиска делится на несколько поисковых слов. Если оно включает какое-то ключевое слово или ключевое слово (длиннее 3 символов) включает поисковое, то его вес страницы увеличивается
4. Сайты сортируются по весу ключевых слов от больших (сверху) к меньшим (снизу)
5. Пока идёт поиск пишется об этом, если не найдёт, тоже будет подсказка об этом.

Как можно изменить этот проект под свои задачи
=====================
1. Измените название страниц под свои нужды
2. В файле **finder.js** измените **pages_to_search** под свои страницы и ключевые слова. Первая надпись будет отображать, как название страницы. Так же вы можете указать **tear** чтобы указывать вес этого ключевого слова
3. В файле **recommendations.js** измените рекомендации под свои
4. Замените ИИ в ai/aiChat.js laboratory/laboratory.js
