Оксиды - Конспект по химии

Что такое оксиды?

Оксиды - это сложные вещества, состоящие из двух элементов, один из которых - кислород в степени окисления -2. Кислород в оксидах проявляет постоянную валентность II. Исключения - пероксиды (например, H₂O₂), где кислород имеет степень окисления -1.

Общая формула оксидов: ЭmOn, где Э - химический элемент. Название оксида состоит из слова "оксид", после которого в родительном падеже указывается название элемента с указанием его степени окисления, если она переменная (например, оксид железа(III) - Fe₂O₃).

Классификация оксидов

Оксиды делятся на несколько типов в зависимости от их химических свойств:

  • Основные оксиды - образованы металлами в низких степенях окисления (+1, +2). Реагируют с кислотами с образованием соли и воды. Пример: Na₂O, CaO. Эти оксиды соответствуют основаниям (Na₂O → NaOH, CaO → Ca(OH)₂).
  • Кислотные оксиды - образованы неметаллами или металлами в высоких степенях окисления (+5, +6, +7). Реагируют с основаниями. Пример: CO₂, SO₃, CrO₃. Эти оксиды соответствуют кислотам (CO₂ → H₂CO₃, SO₃ → H₂SO₄).
  • Амфотерные оксиды - проявляют как кислотные, так и основные свойства. Пример: Al₂O₃, ZnO, Cr₂O₃. Они могут реагировать как с кислотами, так и с щелочами, образуя соли. Например, Al₂O₃ + 6HCl → 2AlCl₃ + 3H₂O и Al₂O₃ + 2NaOH + 3H₂O → 2Na[Al(OH)₄].
  • Несолеобразующие (безразличные) оксиды - не образуют солей при взаимодействии с кислотами или основаниями. Пример: CO, NO, N₂O. Эти оксиды не соответствуют ни кислотам, ни основаниям.

Получение оксидов

Оксиды можно получить различными способами:

  1. Взаимодействие простых веществ с кислородом:
    4Al + 3O₂ → 2Al₂O₃
    S + O₂ → SO₂
    2Mg + O₂ → 2MgO
  2. Разложение сложных веществ (кислот, оснований, солей):
    H₂CO₃ → CO₂ + H₂O
    CaCO₃ → CaO + CO₂
    2Fe(OH)₃ → Fe₂O₃ + 3H₂O
  3. Окисление бинарных соединений:
    2H₂S + 3O₂ → 2SO₂ + 2H₂O
    CH₄ + 2O₂ → CO₂ + 2H₂O
  4. Окисление простых веществ в ограниченном количестве кислорода:
    2C + O₂ → 2CO
    2Cu + O₂ → 2CuO

Химические свойства оксидов

Основные оксиды:

  • С кислотами: CaO + 2HCl → CaCl₂ + H₂O
  • С кислотными оксидами: CaO + CO₂ → CaCO₃
  • С водой (только активные): Na₂O + H₂O → 2NaOH. Неактивные (например, CuO) с водой не реагируют.

Кислотные оксиды:

  • С основаниями: SO₃ + 2NaOH → Na₂SO₄ + H₂O
  • С основными оксидами: CO₂ + CaO → CaCO₃
  • С водой: SO₃ + H₂O → H₂SO₄. Некоторые кислотные оксиды (например, SiO₂) с водой не реагируют, но взаимодействуют с основаниями.

Амфотерные оксиды:

  • С кислотой: ZnO + 2HCl → ZnCl₂ + H₂O
  • С щёлочью: ZnO + 2NaOH + H₂O → Na₂[Zn(OH)₄] (тетрагидроксоцинкат натрия)

Примеры важных оксидов

Формула Название Тип оксида Применение
CO₂ Углекислый газ Кислотный Газировка, пожаротушение, фотосинтез, сухой лёд

Заключение

Оксиды - важнейший класс неорганических соединений, играющий ключевую роль в промышленности, природных процессах и повседневной жизни. Понимание их классификации и свойств необходимо для дальнейшего изучения химии. Особое внимание следует уделять амфотерным оксидам, так как их двойственная природа является важным проявлением периодического закона.
